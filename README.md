# Projeto-LG1A1

Executar o Admin.exe

Projeto em C que consiste em um sitema de Drive Thru.

Conta com os seguintes módulos:

Menu de administração, 
Leitor de itens do menu, 
Cadastrador de itens, 
Vendas, 
Pagamentos e 
Retirada.
